# Windsong Converter
This project converts osu! beatmaps into [Windsong](https://github.com/itdelatrisu/windsong)
format.  It is written in Java.

## Instructions
* Compile the code:
  
  ```
  javac -d . *.java
  ```
* Run the converter:
  
  ```
  java itdelatrisu.windsong.converter.Converter [root_beatmap_folder]
  ```
